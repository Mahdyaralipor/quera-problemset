# quera-problemset

این مخزن شامل حل **سوالات** وب‌سایت [Quera](https://quera.org) است که به‌صورت مرحله‌به‌مرحله اضافه می‌شوند. [web:9]

## هدف

- تمرین و تقویت مهارت‌های برنامه‌نویسی.
- آشنایی بیشتر با الگوریتم‌ها و ساختار داده‌ها از طریق سوالات کوئرا. [web:4]
- داشتن یک آرشیو منظم از کدهای حل شده برای مرور و توسعه‌ی بعدی.

## ساختار مخزن

(این بخش را متناسب با ساختار واقعی ریپو خودت اصلاح کن؛ مثلا اگر برای هر سوال یک فولدر جدا داری:)

- `problems/`  
  شامل پوشه‌های جداگانه برای هر سوال، به‌صورت زیر:
  - `problem-ID/`
    - `solution.ext` (کد اصلی)
    - `README.md` (توضیح کوتاه سوال، لینک کوئرا، و ایده‌ی حل)

مثال نام‌گذاری:
- `problems/589/` برای سوال Quera با شناسه 589.
- `problems/280/` برای سوال Quera با شناسه 280. [web:4]

## نحوه اجرا

1. وارد پوشه‌ی مربوط به هر سوال شوید:
   ```bash
   cd problems/<problem-id>
php solution.php < input.txt
``` [web:7]
